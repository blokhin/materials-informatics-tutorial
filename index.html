crystal_obj = crystal(

        ( 0.235, 0.0, 0.5 ),
        ( 0.5861, 0.0, 0.0 ),
        ( 0.333, 0.666, 0.5 ),
        ( 0.0, 0.0, 0.0 )
    ],
    spacegroup=189,
    cellpar=[7.033, 7.033, 4.023, 90, 90, 120],
    primitive_cell=True
)

  • POSCAR and CIF are the most common formats for crystal structures
  • the modern materials simulations can be prepared fully online
